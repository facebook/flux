튜토리얼 – Chat

이 애플리케이션은 ForwardJS 컨퍼런스를 위해 Flux 앱 구조를 어떻게 잡아야 하는지, 그리고 Dispatcher.waitFor()를 활용해 dispatcher에 등록한 콜백을 올바른 순서대로 실행하게 할 수 있게 만드는지에 관한 예제로 작성하였다.

전체 예제 애플리케이션은 React GitHub 리포지터리의 예제 폴더 내 flux-chat에서 찾을 수 있다.
